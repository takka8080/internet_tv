# internet_tvのデータベースとテーブルの構築及びサンプルデータの挿入手順
## 1. データベースの構築
### 1-1. 構築
```mysql:MySQL
CREATE DATABASE internet_tv;
```
### 1-2. 移動
```mysql:MySQL
USE internet_tv;
```

## 2. テーブルの構築
* usersテーブル
```mysql:MYSQL
CREATE TABLE users(
    user_id BIGINT(20) PRIMARY KEY AUTO_INCREMENT,
    user_name VARCHAR(100) UNIQUE NOT NULL,
    email VARCHAR(100) UNIQUE NOT NULL,
    passward_hash VARCHAR(255) NOT NULL,
    create_at DATETIME NOT NULL
);
```
* channelsテーブル
```mysql:MySQL
CREATE TABLE channels(
    channel_id INT(10) PRIMARY KEY AUTO_INCREMENT,
    channel_name VARCHAR(50) NOT NULL
);
```
* genresテーブル
```mysql:MySQL
CREATE TABLE genres(
    genre_id INT(10) PRIMARY KEY AUTO_INCREMENT,
    genre_name VARCHAR(50) NOT NULL
);
```
* programsテーブル
```mysql:MySQL
CREATE TABLE programs(
    program_id BIGINT(20) PRIMARY KEY AUTO_INCREMENT,
    program_name VARCHAR(50) NOT NULL,
    description TEXT NOT NULL
);
```
* program_genresテーブル
```mysql:MySQL
CREATE TABLE program_genres(
    program_id BIGINT(20) NOT NULL,
    genre_id INT(10) NOT NULL,
    FOREIGN KEY (program_id) REFERENCES programs(program_id),
    FOREIGN KEY (genre_id) REFERENCES genres(genre_id)
);
```
* episodesテーブル
```mysql:MySQL
CREATE TABLE episodes(
    episode_id BIGINT(20) PRIMARY KEY AUTO_INCREMENT,
    program_id BIGINT(20) NOT NULL,
    season_number INT(10),
    episode_number INT(10),
    title VARCHAR(300) NOT NULL,
    description TEXT NOT NULL,
    video_time INT(10) NOT NULL,
    release_date DATE NOT NULL,
    FOREIGN KEY (program_id) REFERENCES programs(program_id)
);
```
* time_slotテーブル
```mysql:MySQL
CREATE TABLE time_slot(
    slot_id BIGINT(20) PRIMARY KEY AUTO_INCREMENT,
    channel_id INT(10) NOT NULL,
    start_time DATETIME NOT NULL,
    end_time DATETIME NOT NULL,
    FOREIGN KEY (channel_id) REFERENCES channels(channel_id)
);
```
* broadcastsテーブル
```mysql:MySQL
CREATE TABLE broadcasts(
    broadcast_id BIGINT(20) PRIMARY KEY AUTO_INCREMENT,
    slot_id BIGINT(20) NOT NULL,
    episode_id BIGINT(20) NOT NULL,
    view_count INT(10) NOT NULL,
    FOREIGN KEY (slot_id) REFERENCES time_slot(slot_id),
    FOREIGN KEY (episode_id) REFERENCES episodes(episode_id)
);
```
* commentテーブル
```mysql:MySQL
CREATE TABLE comments(
    comment_id BIGINT(20) PRIMARY KEY AUTO_INCREMENT,
    user_id BIGINT(20) NOT NULL,
    episode_id BIGINT(20) NOT NULL,
    comment TEXT NOT NULL,
    write_date DATETIME NOT NULL,
    FOREIGN KEY (user_id) REFERENCES users(user_id),
    FOREIGN KEY (episode_id) REFERENCES episodes(episode_id)
);
```
* view_historyテーブル
```mysql:MySQL
CREATE TABLE view_history(
    view_id BIGINT(20) PRIMARY KEY AUTO_INCREMENT,
    user_id BIGINT(20) NOT NULL,
    episode_id BIGINT(20) NOT NULL,
    view_date DATETIME NOT NULL,
    FOREIGN KEY (user_id) REFERENCES users(user_id),
    FOREIGN KEY (episode_id) REFERENCES episodes(episode_id)
);
```
